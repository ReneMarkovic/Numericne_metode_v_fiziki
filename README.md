## Izdelane simulacije

1. Sončni sistem (Reševanje diferencialnih enačb)
   Dinamika sistema $N$-teles (Sonce, Zemlja, Luna, ...). Izračun trajektorij in zapis v rezultati.txt. Numerično preverjanje ohranitve celotne energije in vrtilne količine sistema skozi čas. Raziskovanje vpliva koraka $dt$ na stabilnost Luno-Zemljine orbite.

2. Monte Carlo Molekularna Dinamika
   Termodinamične lastnosti snovi pri določeni temperaturi $T$. Izračun povprečne energije sistema v odvisnosti od temperature $T$.Opazovanje prehoda iz urejene (kristalne) v neurejeno (plinsko) fazo.

3. Analiza časovnih vrst (FFT in spektralna analiza)
   Hitra Fourierova transformacija (FFT). Časovna vrsta iz simulacije planetov (npr. oddaljenost Zemlje od Sonca) ali nihala. Izračun močnostnega spektra (Power Spectrum) za določitev periodičnosti. Identifikacija glavnih frekvenc v signalu in ločevanje fizikalnega signala od numeričnega šuma. Vpeljava filtrov za low-pass, high-pass, band-pass in druge pristope za izboljšanje analize časovnih vrst.

4. Python in tehnike vizualizacije

- **A. Uvod v Python za fizike**: Python kot primarno orodje za znanstveno analizo, ki zaradi svoje knjižnične podpore omogoča hiter prehod od surovih številk do interpretacije.
   
- **B. Povezava C++ simulacij s Pythonom**: Težišče tega sklopa je avtomatizacija analize podatkov, pridobljenih z lastnimi C++ programi.

- **Napredna vizualizacija (Blender)**: Za kompleksne sisteme, kjer 2D grafi ne zadoščajo, študenti uporabijo programsko opremo Blender za 3D upodabljanje.
